# PSKit

PowerShell 7 toolbox with Oh My Posh theming.

## Install

```powershell
irm https://raw.githubusercontent.com/agniuks/PSKit/main/install.ps1 | iex
```

Requires PowerShell 7+ and winget.

## Uninstall

```powershell
Uninstall-PSKit
```

Or: `Uninstall-PSKit -RemoveData` to also delete theme files.

## Commands

- `Get-PSKitStatus` — show what's installed
- `Uninstall-PSKit` — remove PSKit
